## 2025-05-10 — Skip Links and Focus Management in React SPAs
**Learning:** While the project is visually themed as a WordPress CMS with Neo-Editorial aesthetics, it's a React Single Page Application (SPA). The standard WordPress `.screen-reader-text` utility wasn't present, so I implemented a standard `.skip-link` accessibility pattern natively with CSS and HTML properties. Managing focus target effectively required adding `tabIndex={-1}` and `outline: "none"` to the main content container to ensure proper routing from the skip link without triggering default outlines.
**Action:** When applying WordPress accessibility paradigms like skip-to-content links to React SPAs, ensure the target element (usually `<main>`) has a programmatically manageable `tabIndex={-1}` to receive focus correctly after a hash navigation event.
